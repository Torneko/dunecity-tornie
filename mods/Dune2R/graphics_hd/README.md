# Dune2R Enhanced Graphics

Place high-resolution anchored graphics here.

Expected first layout:

```text
units/
  OrdosFatTank/
    unit.json
    idle/
    movement/
    combat/
```

Enhanced assets are intended for a render overlay. They can be larger than the
classic tile art, but each unit needs metadata that defines its anchor point,
direction order, and supported states.
